Tech Industry Company Analysis and Selection

Python Pandas NumPy Data Analysis

Overview

This project involved a comprehensive analysis of companies in the tech industry to identify the most financially stable and promising candidates for a Tech Recruiting company. The process included industry identification, data processing, and advanced analysis techniques.

Key Steps

  1. Research and Target Industry Identification
  2. NACE Code Selection for tech industry
  3. Data Processing and Analysis using Python (Pandas, NumPy):
    • Initial filtering based on NACE codes
    • Data normalization of financial indicators
    • Ranking and selection of top companies
  4. Output Generation:
    • Highlighted selected companies in the original file
    • Created a separate sorted list of chosen companies

Conclusion

This project demonstrated a data-driven approach to identifying promising tech companies, ensuring a balanced consideration of various financial indicators while focusing on the relevant industry sector.

Copart Lot Price Analysis Tool

Python Flask Selenium HTML/CSS

Overview

The Copart Lot Price Analysis Tool is a web-based application designed to assist individuals who buy vehicles on Copart, repair them, and sell them for profit. By automating the process of gathering and analyzing vehicle prices, this tool saves time and provides valuable insights into the potential profitability of a purchase.

Key Features

  • Input field for Copart lot number
  • Automated data gathering from Copart
  • Price comparison with Georgian car-selling platform Myauto
  • Calculation of minimum, maximum, and mean prices

Drug Price Comparison Tool

Python Flask Selenium HTML/CSS

Overview

The Drug Price Comparison Tool is a web-based application that helps users find the most optimal price for medications available across four major online pharmacies in Georgia: Aversi, PSP, GPC, and Pharmadepot. By entering the name of a drug in the search field, users can instantly compare prices and other relevant details, such as the country of origin and available images, all on a single page.

Key Features

  • Search functionality across multiple pharmacies
  • Automated price comparison and sorting
  • Detailed information display including drug origin and images
  • Direct links to pharmacy pages for easy purchasing
